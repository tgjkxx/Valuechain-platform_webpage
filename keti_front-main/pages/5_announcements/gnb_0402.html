자주하는 질문

데이터 신청 자격은?
플랫폼은 개인정보를 철저히 보호하고자 데이터 신청자를 대한민국 국적의 국내거주자로 제한하고 있습니다.
데이터를 신청하고 활용하는데 제한은 있나요?
데이터 신청은 대한민국 국적의 국내거주자이면 누구에게나 연구를 수행하기 위하여 데이터를 신청할 수 있습니다.
다만, 고의적으로 데이터 유출이나 악의적 목적으로 데이터를 신청할 경우에는 민형사상의 법적인 책임을 지게 됩니다
데이터 카탈로그에서 제공되는 목록 이외의 데이터는 신청할 수 없나요?
플랫폼에서 제공되는 데이터는 데이터 카탈로그를 기준으로 하고 있으며, 데이터 카탈로그정보는 지속적으로 업데이트할 예정입니다.
플랫폼을 활용한 연구결과를 외부에 발표할때에는 출처를 명기해야하나요?
플랫폼 데이터를 활용한 연구결과를 외부에 발표할때에는 데이터 출처를 본 밸류체인 헬스케어 데이터 공유 플랫폼이라고 명기해야합니다.
