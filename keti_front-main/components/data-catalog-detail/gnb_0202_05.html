WIDER FACE: A FaceDetection Benchmark

얼굴 검출 벤치마크 데이터

데이터 개요
  • WIDER FACE 데이터세트는 얼굴 검출 벤치마크 데이터세트로, 공개된 WIDER 데이터세트에서 이미지를 선택했습니다.
  • 32,203개의 이미지를 선택하고 샘플 이미지에 표시된 대로 크기, 포즈 및 폐색의 가변성이 높은 393,703개의 얼굴에 라벨을 지정했습니다.
  • WIDER FACE 데이터 세트는 61개의 이벤트 클래스를 기반으로 구성됩니다.
  • 각 이벤트 클래스에 대해 훈련, 검증 및 테스트 세트로 40%/10%/50% 데이터를 무작위로 선택합니다.
  • PASCAL VOC 데이터 세트에 사용된 것과 동일한 평가 지표를 채택합니다.
데이터 구조
및 형식
  • The format of txt ground truth.
  • File name
  • Number of bounding box
  • x1, y1, w, h, blur, expression, illumination, invalid, occlusion, pose
